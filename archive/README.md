# Archive

Fichiers archivés, conservés pour référence mais non utilisés par l’application.

## Contenu

- **`bak_routes/`** : anciennes sauvegardes de routes (panier, admin) avant refactor / migration Vue.
- **`bak_views/`** : anciennes sauvegardes de vues EJS (admin dashboard, catalogues, produits, catégories, fournisseurs, etc.) avant migration vers Vue/Vite.
- **`vue_cdn/`** : anciennes apps Vue CDN (AdminProduitsApp.js, PanierApp.js, HomeApp.js, etc.) et composants associés, remplacées par les bundles Vite (`public/dist/`). Le script `responsive-helper.js` a été déplacé dans `public/js/` et reste chargé depuis le header.

Ces fichiers peuvent être supprimés définitivement si plus aucune référence n’est nécessaire.
